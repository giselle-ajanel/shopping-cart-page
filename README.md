<h2>Dynamic Shopping Cart App</h2>
Hi there! Welcome to this project.

This is a fully functional, interactive Shopping Cart web application that manages items in real time. I built this to dive deep into vanilla JavaScript DOM manipulation and form validation, styling the user interface cleanly using the Bootstrap 4 framework.
The challenge I set for myself was creating a seamless, interactive user experience entirely on the front end handling real-time item creation, duplicate prevention, and item removal without relying on complex frameworks like React.

<h2>Tech & Design</h2>

Vanilla JavaScript DOM Engine: Programmed dynamic document node creation (document.createElement) and tree traversal (parentElement) to inject or remove list elements on the fly.

Smart Form Validation: Built custom real-time error handling that intercepts form submissions to prevent empty inputs, handle missing data gracefully, and cross-reference a state check to block duplicate items from being added.

Bootstrap 4 Layout: Styled with Bootstrap's responsive grid system and utility classes to ensure the form and the cart list adjust smoothly across desktop and mobile screens.
